# WhatsApp SD Redirect - Módulo LSPosed

Redirige automáticamente archivos de WhatsApp a microSD con detección automática.

## Instalación
1. Compila: `./gradlew assembleDebug`
2. Instala el APK en tu móvil
3. Activa en LSPosed Manager
4. Marca scope: com.whatsapp
5. Reinicia el dispositivo

## Verificación
Revisa los logs en LSPosed Manager para ver la SD detectada:
- LSPosed Manager → Logs
- Busca: [WhatsAppSDRedirect]
